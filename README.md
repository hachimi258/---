# 知到智慧树自动输入工具

->！！！windows和python！！！
-> 原理：模拟键盘输入，将 `zhidao.txt` 中的内容逐个字符发送到当前光标位置。  
> 适合需要重复输入大段文本的场景（如智慧树平台答题）。

## 文件说明
- `setup.py`   – 自动部署脚本（检测 Python、安装依赖、生成配置文件）  
- `run.bat`    – 启动脚本（请求管理员权限，运行主程序）  
- `zhidao.py`  – 主程序（监听热键，读取 txt 并模拟输入）  
- `zhidao.txt` – 存放待输入的内容（UTF-8 编码）

## 快速开始

### 1. 首次使用：运行 `setup.py`
- 双击 `setup.py`（或在终端执行 `python setup.py`）  
- 脚本会自动：
  - 检测 Python 环境
  - 安装 `keyboard` 依赖（清华源）
  - 创建示例 `zhidao.txt`
  - 配置 `run.bat` 中的 Python 路径

### 2. 编辑要输入的内容
- 打开 `zhidao.txt`，删除示例文字，写入你需要自动输入的内容
- 保存文件（`Ctrl+S`）

### 3. 启动工具
- 双击 `run.bat`（会弹出 UAC 请求管理员权限，请同意）  
- 保持黑色终端窗口**最小化运行，不要关闭**

### 4. 输入内容
- 点击任意输入框（如浏览器的文本区、答题框）
- 按下快捷键（默认 <kbd>~</kbd> 键，即数字 `1` 左边的按键）
- 文件中的内容会像人工打字一样逐字输入
